# 穿戴甲客户信息采集 SOP

> 每接触一个新客户，按这个流程走一遍，信息不漏、报价不乱、客户不丢

---

## 💰 核心思维：不只赚树脂的钱

| 赚什么 | 具体 | 频率 |
|------|------|:---:|
| 树脂 | 打穿戴甲的专用树脂 | 每月复购 |
| 设备 | 打印机、UV固化箱、超声波清洗机 | 一次性 |
| 耗材包 | FEP膜、IPA、手套、打磨耗材 | 每月复购 |
| 辅料包 | 果冻胶、包装卡纸、吸塑托盘 | 每月复购 |
| 方案 | 设备选型 + 参数调试 + 培训 | 一次性 |

> **你的角色不是"卖树脂的"，是"穿戴甲生产整包方案商"。**
> 客户要的不是一桶树脂，是一套能跑通的产线。

---

## 第一步：破冰后挖需求

### 微信版（国内客户直接用）

```
老板，做穿戴甲的话我先了解一下你的情况，这样我好给你配方案：

1. 现在在做了吗？还是准备上？
2. 主要做国内市场还是出口？走什么渠道？
3. 大概一个月能走多少套？
4. 想做什么风格？纯色/渐变/浮雕/3D立体？
5. 尺码是做标准码还是定制扫描？
6. 包装要带自己品牌吗（OEM）？
7. 设备方面现在有什么？还缺哪些？

不用全回，大概说一下就行，我帮你配 👊
```

## 第三步：不同客户，不同套餐

### 套餐A：完全没开始的

> "老板你这一块还没上设备的话，我给你配一套**新手启动包**"

| 项目 | 内容 | 赚什么 |
|------|------|:---:|
| 打印机 | 推荐2-3款适配型号 | 设备差价/佣金 |
| UV固化箱 | 跟树脂配套推荐 | 设备 |
| 超声波清洗机 | 配套 | 设备 |
| 穿戴甲专用树脂 | 首批5-10kg试机 | 树脂 |
| 耗材新手包 | FEP膜+手套+IPA+打磨工具 | 耗材 |
| 参数调试支持 | 远程/上门调参数 | 方案（免费→锁客） |

### 套餐B：有设备，想扩产的

> "设备你有了，那我主要帮你搞定**树脂+耗材稳定供应**这块"

| 项目 | 内容 | 赚什么 |
|------|------|:---:|
| 穿戴甲专用树脂 | 批量价 | 树脂（核心利润） |
| 耗材月包 | FEP膜+手套+清洁耗材，按月配 | 耗材月费 |
| 辅料月包 | 果冻胶+卡纸+吸塑托盘 | 辅料月费 |
| 新色开发 | 按需调色 | 增值服务 |

### 套餐C：已经在量产，想降本/提效的

> "你现在量跑起来了，我主要帮你**把成本打下来，效率提上去**"

| 项目 | 内容 | 赚什么 |
|------|------|:---:|
| 大桶树脂（10kg+） | 阶梯价，量越大越便宜 | 树脂走量 |
| 省耗材方案 | 换长寿命FEP膜、IPA回收方案 | 耗材升级 |
| 废液回收方案 | 固化+回收，减少IPA消耗 | 方案增值 |
| OEM辅料定制 | 包装印自己logo | 辅料定制 |

## 第四步：客户分级

| 级别 | 特征 | 下一步 |
|:---:|------|------|
| 🥇 A | 已经在做，有量，想扩 | 寄样试机 → 报价 → 签单 |
| 🥈 B | 准备上，设备还没买 | 推启动包 → 锁设备+树脂绑定 |
| 🥉 C | 还在看，没有明确时间 | 发案例 + 拉群 + 每月跟进 |

## 第五步：话术库（中文）

### 第一次聊

```
老板你好！我们 Resione 在3D打印树脂这块做了很多年了，穿戴甲这个赛道最近跑得很快。你这边什么情况？已经在做了还是准备上？
```

### 挖需求

```
你现在设备是哪些？打印机、固化机、清洗机都配齐了吗？树脂用的是谁家的？用着顺手不？
```

### 推整包

```
其实单买树脂是一回事，我是建议你耗材整体打包，我给你按月配好，FEP膜、手套、清洁耗材这些不用你操心，准时送到，价格还比零买便宜。
```

### 报价后

```
老板，方案发你了。你看一下，这个价格是包含了树脂+耗材月包的，比你单买省 [X]%。有哪里需要调的直接说。
```

### 催回复

```
老板，上次的方案看了没？不急，就是怕你那边项目有变动，随时找我哈 🙏
```

### 长期维护

```
老板最近量怎么样？树脂快用完了吧，这批要不要提前给你备上？省得到时候等货耽误生产。
```

## 🎯 核心原则

1. **你不是卖树脂的，你是卖产线的** — 一单=树脂+设备+耗材+辅料
2. **第一次聊就挖全需求** — 不问清楚就报价，后面全是坑
3. **耗材锁定 = 客户跑不掉** — 月包模式大于零散采购
4. **分级追** — A紧追、B推包、C养着
5. **成交不是结束，是复购的开始** — 每月问一次库存
