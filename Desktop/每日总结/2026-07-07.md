## 🎯 穿戴甲新项目调研

> 这是今天的核心产出。3D打印穿戴甲作为 Resione 新业务线，做"整包方案商"——不只卖树脂，卖整条产线。

### 先例盘点

| 公司 | 做什么 | 与我们的差异 |
|------|------|------|
| 广东创新科学技术 | C端门店+AI定制+故宫文创IP | 我们做B端整包 |
| Xianxing Beauty | OEM注塑日产30万片 | 我们走3D打印柔性路线 |
| Senboma | 年产能1.5亿片注塑 | 我们做中高端定制 |
| Fyne Body（澳洲） | UV打印定制 | 我们供产线不是卖成品 |
| NewNails（新加坡） | 扫描+AI+光固化 | 学院项目，未商业化 |
| 富甲天下（天津） | 穿戴甲批量打印托专利 | 做设备端 |

**核心判断：市场空白——没人做"穿戴甲生产整包方案商"。大家都在卖甲片，我们在卖"造甲片的能力"。**

### 生产流程（7大环节）

```
① 数据采集 → ② 建模设计 → ③ 3D打印 → ④ 后处理 → ⑤ 上色装饰 → ⑥ 质检 → ⑦ 包装出货
```

### 后处理是关键

```
清洗 → UV二次固化（必须完全固化！）→ 去支撑 → 修边 → 打磨 → 上色→ 封层 → 贴果冻胶
```
> 80%的问题出在后处理，不是打印。漏一步 = 退货/过敏。

### 三种量产方案

| 方案 | 打印负责 | 上色负责 | 适用 |
|------|------|------|------|
| A 纯打印 | 打白模 | 全手工 | 高端定制 |
| B 打印+UV喷印 | 打白模 | UV平板机喷图案+手工封层 | **量产首选** |
| C 全打印 | 一体打印含浮雕 | 手工润色 | 重工款 |

### 材料安全（底线）

- 普通光固化树脂 ≠ 皮肤安全 → 必须用 ISO 10993 认证的皮肤安全级树脂
- 未完全固化 = 残留单体 = 永久性过敏风险
- 必须二次固化 + 封层胶隔绝

### 镜面效果科普

- 原理：镜面粉（铬粉）微米金属颗粒 + 极致光滑表面 = 镜面反光
- 流程：黑底胶 → 半干免洗封层 → 擦镜面粉 → 再封层
- 3D打印甲片能做，底子越光滑效果越好

### F69 vs F80 手机壳材质

- F80（~80A硬度）更适合做手机壳主体
- F69（~69A）偏软，只适合按键/包角等局部应用
- 手机壳甜点硬度：80A-85A

### 树脂设备兼容性

- Resione 树脂适配 405nm 波段，LCD/DLP 通用
- 不同波段需确认技术文档（385nm 牙科设备）

### ABS 在穿戴甲中的角色

- ABS 是传统注塑路线（一副成本2-5元，走量）
- 光固化树脂是差异化路线（柔韧/贴合/质感/定制化）
- 不要跟 ABS 拼价格，拼的是利润更高的那端
