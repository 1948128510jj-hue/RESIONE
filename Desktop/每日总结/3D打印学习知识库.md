## 一、打印机硬件知识

### 核心部件

| 部件 | 作用 | 注意事项 |
|------|------|----------|
| 构建板/平台 | 打印件粘附的底板 | 只抓最上端，倾斜取出，不泡酒精 |
| FEP 离型膜 | 料槽底部的透明膜 | **极其脆弱**，不能用纸擦，只走无纺布 |
| 屏幕 | 光固化成像 | 酒精是溶剂不能直接接触，会短路烧屏 |
| 料槽 | 装树脂的容器 | 先清平台再清料槽，避免滴液到屏幕 |

### 关键参数

```
层厚：0.05mm（精细）/ 0.1mm（快速）
底部层数：4 层
底部曝光：25-35s（按树脂调）
正常曝光：2.5-3.2s（按树脂调）
Z轴高度：一般 5mm，贴地打（0mm）= 实验用
```

### 气泡处理

```
1. 树脂倒入料槽后静置片刻让气泡上浮
2. 塑料铲沿边缘轻轻刮，把气泡推到边边
3. 确认无可见气泡后再打印
→ 气泡不弄干净 = 打印件表面有坑洞/缺陷
```

### RESIONE 11 大类别

### 命名规则

```
TH- = Tough & Hard（坚韧系列）
M   = 单色坚韧（M58灰/M68白/M70红蜡）
K   = 黑色坚韧（K, K+）
F   = 柔性（F69/F39/F39T/F80）
WW  = 水洗（WW123/WW-ABS）
D   = 牙科（D01/D01S）
SP  = 标准Pro
G   = 透明（G217）
```

### 关键性能指标解读

| 指标 | 含义 | 越高? |
|------|------|------|
| 硬度（Shore D/A） | D=硬质，A=软质 | 刚性↑ 脆性↑ |
| 拉伸强度（MPa） | 拉断需要多大力 | 强度↑ |
| 断裂伸长率（%） | 拉多长才断 | 韧性↑ |
| 弯曲强度/模量（MPa） | 抗弯能力 | 刚性↑ |
| 缺口冲击（J/m） | 抗冲击能力 | 耐摔↑ |
| 吸水率（%） | 24小时吸水百分比 | 越低越好（非水洗） |

### 力学性能深层认知（6/13-14 周末学习）

#### 拉伸强度 (Tensile Strength)
- 光固化打印件的拉伸强度通常只有注塑件的 60-80%
- XY 方向强度接近材料本体，Z 方向（层间）是薄弱环节
- 关键影响因素：填充率 > 打印温度 > 层高 > 打印速度

#### 弯曲模量 (Flexural Modulus)
- 壁厚（shell count）对弯曲刚度的贡献远大于填充率
- 退火处理可提升弯曲模量 10-20%

#### 冲击韧性 (Impact Resistance)
- 提高打印温度 + 降低冷却风扇 = 更好的层间结合 = 更高冲击韧性
- 层间结合力决定了 Z 向冲击性能

#### 层间结合力 (Interlayer Adhesion)
- 这是光固化打印最关键的力学短板
- 热退火（annealing）可部分修复层间缺陷
- 树脂本身的交联密度决定层间结合上限

#### 力学指标矛盾关系（核心认知）

```
刚性 ⇄ 韧性（不可兼得）
硬度 ↑ → 伸长率 ↓ → 脆
冲击 ↑ → 硬度 ↓ → 软
精度 ↑ → 通常脆性 ↑
耐热 ↑ → 通常脆性 ↑（如 HT-Enduse 97D 但冲击仅 15 J/m）
吸水率与水洗不可兼得（水洗=高吸水）
```

**如何选材：** 先确定第一优先级（刚性？韧性？耐热？），其他指标接受妥协。

#### 吸水率分级

| 级别 | 吸水率 | 代表产品 | 适用场景 |
|------|:--:|------|------|
| 极低 | <0.5% | HT-Enduse(0.02%), K+(0.30%), TH-MINI(0.40%) | 户外/水下短期 |
| 中等 | 0.5-1.5% | 大多数坚韧树脂 | 室内良好 |
| 高 | >5% | WW123(5.71%), TH-WW(16.7%) | 水洗树脂正常，避开水环境 |

#### LCD 树脂力学性能分级（0-10 评分体系）

| 等级 | 分数 | 含义 |
|:--:|:--:|------|
| 优异 | 9-10 | 该性能顶级水平 |
| 良好 | 7-8 | 满足大多数需求 |
| 中等 | 5-6 | 可接受，有限制 |
| 较差 | 3-4 | 明显短板 |
| 很差 | 1-2 | 不适合该场景 |

> 注：以上数据均由 RESIONE 实验室在湿度 50-60%、温度 25°C 环境下测得，仅对 RESIONE 品牌有效，与其他品牌对比无意义。

### 打印特性认知（FDM/光固化通用）

#### 环境因素
- 温度：低温环境（<20°C）树脂变稠，影响流动性，韧性下降
- 湿度：高湿环境加速树脂吸湿，尤其水洗系列
- 冬天 vs 夏天：冬天树脂更脆，需要适当提高曝光时间

#### 第一层的重要性
- 第一层是成败关键 — 构建板调平（Z-offset）校准最重要
- 底部曝光一定要够（25-35s），宁可过不可欠
- 构建板附着：PEI > 纹理玻璃 > 平滑玻璃

#### 树脂存储
- 避光保存（UV 会固化树脂）
- 密封防潮
- 用前摇匀（填料可能沉淀）
- 瓶口用完擦干净（干了封死）

### 选材决策树

```
打印什么？
├─ 手办/GK → M58/M70/TH-HR
├─ BJD娃娃 → TH-BJD
├─ 战棋微缩 → TH-MINI
├─ 功能零件 → Tough74 V2/K+/M58
├─ 透明件 → G217/F39T
├─ 软胶件 → F69/F39/F39T
├─ 浴室/户外 → K+/CL-TH（低吸水）
├─ 高温环境 → HT-Enduse
├─ 日常练习 → WW123
└─ 牙科模型 → D01/D01S
```

### 水洗树脂特别注意

```
✅ 可用水清洗，无需酒精
❌ 不能放浴室/水边（吸水率 5-17%）
❌ TH-WW 只有灰/火焰橙，没有透明色
✅ WW123 6色可选（含水晶透）
```

## 三、支撑策略体系

### 核心原则

```
1. 正面零支撑 → 脸/前胸/正面细节一根撑不要
2. 红色区域 100% 覆盖 → 悬空岛必撑
3. 细长件加密 → 手指/飘带/毛发/武器
4. 大平面球头撑 → 拆得干净
```

### 摆放是第一决定因素

```
手办类：侧躺 35-45°，脸朝上/朝前，背面朝构建板
功能件：底面大平面贴板，孔朝上
细长件（金箍棒）：斜放 15-85°，不要完全垂直于板
```

### 支撑参数（M58 等坚韧树脂）

| 参数 | 推荐值 |
|------|--------|
| 密度 | 65% |
| 接触形状 | 球形（Sphere） |
| 接触深度 | 0.3mm |
| 顶部直径 | 0.4-0.5mm（脸周围 0.25mm） |
| 中段直径 | 1.2mm |
| 底部直径 | 2.5mm |

### Raft 判断

```
多个独立支撑脚 → 加 Raft（如蝎子8条腿）
单个大底面 → 不需要 Raft
Raft 类型：Cross 网格型（排液更好）
Raft 厚度：1.0-1.2mm
```

### 按树脂类型调支撑

| 树脂类型 | 支撑策略 |
|------|------|
| 刚性（M70/SP64/HT-Enduse） | 细支撑，脆断干净 |
| 坚韧（M58/K+/Tough74 V2） | 中号，韧性好不崩口 |
| 类尼龙（Anti-Impact/TH-MINI） | 加粗一号，高韧性会拉弯细撑 |
| 水洗（WW123/TH-WW） | 加密 10%，水洗树脂稍软 |
| 柔性（F69/F39/F80） | 粗撑+加密，自重会变形 |

### 模型难度分级

```
⭐       底座/方块/牙刷架 — 几乎不思考
⭐⭐     胸像/厚壁道具 — 自动支撑够
⭐⭐⭐   手办（悟空）— 飘带/手指/毛发
⭐⭐⭐⭐ 多件+细长武器 — 每件独立策略
⭐⭐⭐⭐⭐ 蝎子（50万面+8腿+2钳+弯尾）
```

## 四、打印全流程 Checklist

```
□ 1. 开机 → 检查各部件清洁、拧紧
□ 2. 倒树脂入一次性杯 → 擦干净瓶口 → 盖回
□ 3. 倒入料槽 → 静置 → 塑料铲刮气泡
□ 4. 选树脂 → 根据用途选，不是随便拿
□ 5. 导入模型 → 摆放（脸朝上，背朝板，35-45°）
□ 6. 自动支撑 65% + 中
□ 7. 手动检查：脸零支撑、红区全覆盖、飘带/手指补撑
□ 8. Raft？多独立脚=加，单底面=不加
□ 9. 切片预览，检查第 10-30 层
□ 10. 开打
□ 11. 打印完 → 先洗平台再洗料槽
□ 12. 拆撑：刚性直接拆，韧性温水泡30秒
□ 13. 成品判定：干爽不粘 = ✅
```

## 五、清洗与后处理

### 清洗铁律

```
手套不交叉 → 平台不滴液 → 酒精不直喷 → 成品不粘手
```

### 清洗顺序（不能错）

```
先洗平台 → 再洗料槽
原因：平台不干净滴液到屏幕 = 屏幕报废
```

### 各环节要点

| 环节 | 正确做法 | 禁止 |
|------|----------|------|
| 手套 | 两套，脏净分开 | 交叉使用 |
| 屏幕擦拭 | 酒精喷到无纺布上再擦 | 酒精直接倒屏幕上 |
| 平台 | 倾斜拿，只抓上端 | 泡酒精（掉漆） |
| FEP膜 | 无纺布轻柔擦 | 纸巾（掉屑）/用力 |
| 成品验收 | 干爽不粘 | 黏的就继续洗 |

### 拆支撑技巧

```
刚性树脂（SP64/M70）→ 直接掰，脆断干净
坚韧树脂（M58/K+）→ 温水 40-50°C 泡 30 秒再拆
水洗树脂（WW123/TH-WW）→ 温水泡 1 分钟
支撑坑处理 → 800 目砂纸轻磨 → 消光处理
```

## 六、踩过的坑

### 🔴 严重事故

| 事故 | 原因 | 代价 | 预防 |
|------|------|------|------|
| 屏幕烧毁 | 树脂混入酒精渗漏到屏幕 | 换屏幕 | SOP：酒精只走无纺布 |
| 支撑超出构建板 | 没检查投影边界 | FEP膜扯烂 | 切片前确认全覆盖 |

### 🟡 小坑

| 问题 | 教训 |
|------|------|
| 瓶口树脂干结封死 | 倒完必须擦瓶口 |
| 气泡导致表面坑洞 | 塑料铲刮边边 |
| 新设备不会用 | 先问人，不私自盲目操作 |
| TH-WW 说明写错 | 产品数据要交叉验证 |

## 七、软件操作

### CHITUBOX 关键操作

```
导入模型 → 摆放角度
自动支撑 → 密度 65% → 选中
手动编辑支撑 → 脸周围删掉
切片预览 → 检查 10-30 层
导出 .ctb 文件 → 打印
```

### 快捷键/技巧

```
Ctrl+Shift+P → 剖面视图（检查支撑是否穿透正面）
悬空分析 → 红色=必须撑
支撑体积显示 → 必须 >0 才算生效
缩放 → 锁比例链 → 输入百分比
```

## 八、心态与原则

```
1. 细节决定成败 — 每一步都要检查，不能"差不多"
2. 新设备先问人 — 每台打印机脾气不一样
3. 先建 SOP 再操作 — 不靠记忆靠流程
4. 从简单模型开始 — 别一上来就蝎子/龙
5. 失败是学习 — 屏幕烧了 → 写出 SOP → 不会再烧
6. 打印前看官方教程 — 公司抖音号有讲解
```
