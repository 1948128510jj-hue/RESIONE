## 🖨️ 3D打印实操心得

### 支撑策略核心原则

1. **正面优先原则**：支撑永远加在不可见面（背面/底面/内侧），正脸/前胸零支撑。手办打印的灵魂就是"脸不能有坑"。

2. **摆放决定一切**：
   - 手办类：侧躺35-45°，脸朝上/朝前，背面朝构建板
   - 功能性物品（牙刷架）：底面大平面直接贴板，孔朝上
   - 摆放角度比支撑参数更重要 — 摆对了撑自然少

3. **Raft 判断标准**：
   - 多独立支撑脚（如蝎子8条腿）→ 必加Raft
   - 单一大底面 → 不需要Raft
   - 加Raft多花几毛钱树脂，换一次成型成功

### 模型难度分级

| 难度 | 典型模型 | 关键挑战 |
|:--:|------|------|
| ⭐ | 底座/方块/牙刷架 | 几乎不需要思考 |
| ⭐⭐ | 胸像/厚壁道具 | 自动支撑就够了 |
| ⭐⭐⭐ | 手办（悟空/蝎子） | 飘带/手指/毛发/武器 |
| ⭐⭐⭐⭐ | 多件组合+细长武器 | 每件独立策略 |

### 树脂选择决策树

```
问：打印什么？
├─ 手办/GK/精细模型 → M58(灰)/TH-HR(陶泥)/M70(红蜡)
│  └─ 要上色 → M70 鲑鱼色哑光黏土质感
│  └─ 不上色 → M58 灰色/TH-BJD 奶油白
├─ 功能性零件（需要韧性）→ Tough74 V2/Anti-Impact
├─ 透明件 → G217（高透坚韧）/F39T（透明柔性）
├─ 浴室/户外 → K+(0.30%吸水)/CL-TH(0.29%吸水)/M68(白色)
│  └─ 绝不能用水洗树脂！（吸水率>5%）
├─ 弹性/软胶 → F69/F39/F80/GM01
└─ 日常练习/教学 → WW123（12mPas近水流，高速打印）
```

### 今天踩过的坑

1. **TH-WW没有透明色**：产品册OCR描述有误导，"透明动漫特效件"指的是可以用透明树脂打印特效件（如G217），不是TH-WW本身有透明色。TH-WW只有灰/火焰橙两色。

2. **飘带是手办最难的点**：
   - 薄+弯+长 = 三者叠加
   - 解法：让飘带尽量平行于屏幕（截面变化最小）
   - 支撑只插边缘，不插正面
   - 转弯外侧加一根防崩

3. **蝎子不要轻易挑战**：50万面，8腿+2钳+弯尾，属于最高难度。新手从胸像/建筑/厚壁道具开始建立信心。

4. **支撑超出一毫米都不行**：构建板外是FEP膜，支撑跨出去会扯烂膜（换膜¥50-100）或撞坏屏幕。

### 打印流程 checklist

```
□ 1. 选树脂（根据用途，不是随便拿）
□ 2. 模型摆放（脸朝上，背面朝板，35-45°倾斜）
□ 3. 自动支撑 65%密度 + 选中
□ 4. 手动检查：脸周围零支撑、红区全覆盖、飘带/手指补撑
□ 5. Raft？多独立脚=加，单底面=不加
□ 6. 切片预览，检查第10-30层
□ 7. 开打
□ 8. 拆撑：刚性树脂直接拆，韧性树脂温水泡30秒再拆
```
