## 一、力学性能核心指标

### 五个关键指标

| 指标 | 英文 | 含义 | 高 → 低的影响 |
|------|------|------|------|
| 硬度 | Hardness | 抵抗表面压痕的能力 | Shore D=硬质，Shore A=软质 |
| 拉伸强度 | Tensile Strength | 拉断需要多大力（MPa） | 越高越难拉断 |
| 断裂伸长率 | Elongation at Break | 拉多长才会断（%） | 越高越韧 |
| 弯曲强度 | Flexural Strength | 抗弯能力（MPa） | 越高越难弯折 |
| 冲击强度 | Izod Impact | 抗冲击能力（J/m） | 越高越耐摔 |

### 光固化打印的力学短板

```
最大的弱点：Z 轴方向（层间结合力）

每层 0.05mm 的固化层之间是薄弱环节
→ XY 方向强度 ≈ 材料本体的 60-80%
→ Z 方向更弱，冲击时容易分层断裂

怎么补强？
→ 提高曝光时间（增加交联密度）
→ 热退火处理（可提升弯曲模量 10-20%）
→ 选伸长率高的树脂（层间应力分散更好）
```

## 二、力学性能的矛盾关系（核心认知）

### 不可兼得的 Trade-off

```
           刚性
           /\
          /  \
         /    \
        /      \
       /        \
      /  ！！！   \
     / 此消彼长    \
    /              \
  韧性 ———————— 耐热
    \              /
     \            /
      \          /
       \        /
        \      /
         精度
```

### 具体矛盾

| 你想要 | 你就得接受 |
|------|------|
| 刚性高（97D） | 脆，冲击低（HT-Enduse 仅 15 J/m） |
| 韧性高（伸长率 >50%） | 软，硬度低（Anti-Impact 79D） |
| 耐高温（140°C） | 极脆，伸长率仅 2.8%（HT-Enduse） |
| 水洗方便 | 吸水率极高（TH-WW 16.7%），怕水环境 |
| 高精度/细节锐利 | 通常偏脆 |

### 选材第一问

```
这个零件的第一优先级是什么？
├─ 耐摔 → 牺牲刚性
├─ 刚性结构 → 牺牲韧性
├─ 不怕水 → 牺牲水洗便利
├─ 高细节 → 牺牲韧性
└─ 低成本高速 → 牺牲性能
```

## 三、吸水率：被忽视的关键指标

### 为什么重要

```
吸水率高 → 放在潮湿环境 → 尺寸变化 → 力学性能下降 → 零件失效
```

### RESIONE 产品吸水率三级

| 级别 | 吸水率 | 产品 | 能用在哪 |
|------|:--:|------|------|
| 🟢 极低 | <0.5% | HT-Enduse(0.02%), K+(0.30%), TH-MINI(0.40%) | 户外/水下短期 |
| 🟡 中等 | 0.5-1.5% | M58(1.25%), SP64(0.61%), Tough74 V2(0.96%) | 室内正常环境 |
| 🔴 高 | >5% | WW123(5.71%), TH-WW(16.7%) | 水洗树脂，不能碰水环境 |

> 水洗树脂吸水率高是正常的，不是质量问题。但用途受限——绝对不能放浴室/户外。

## 四、LCD 树脂性能评分体系

RESIONE 产品册用 0-10 评分来对比产品各项性能：

| 分数 | 含义 | 举例 |
|:--:|------|------|
| 9-10 | 顶级 | Anti-Impact 冲击韧性 10 分 |
| 7-8 | 优秀 | M58 综合力学 8 分 |
| 5-6 | 中等 | 大多数产品在中高区间 |
| 3-4 | 偏弱 | 刚性树脂的韧性通常 3-4 分 |
| 1-2 | 很差 | 不适合该场景 |

> ⚠️ 这个评分仅对 RESIONE 品牌有效，跟别家不能直接对比。测试条件：湿度 50-60%，温度 25°C。

## 五、环境因素对打印的影响

| 因素 | 影响 | 应对 |
|------|------|------|
| 🌡️ 温度 <20°C | 树脂变稠，流动性差，韧性下降 | 冬天适当提高曝光，或预热树脂 |
| 🌡️ 温度 >35°C | 反应太快，可能过曝 | 降低曝光时间 |
| 💧 高湿度 | 树脂吸水加速（尤其水洗系列） | 密封保存，打印完及时处理 |
| 🌬️ 通风 | 影响不大，但酒精清洗要通风 | 戴口罩手套 |

## 六、树脂存储与处理

```
□ 避光保存（UV 会固化树脂）
□ 密封防潮（干燥剂是好友）
□ 用前摇匀（无机填料会沉淀，如 M70/D01S）
□ 瓶口用完立刻擦干净（残留干了封死瓶盖）
□ 树脂倒入一次性杯再用，不要把料槽里的倒回瓶里
□ 不同树脂不能混（不相容会分层/堵喷头）
```

## 八、最大收获

> **没有完美的树脂，只有匹配的树脂。**
>
> 刚性和韧性天生矛盾，耐热和柔韧不可兼得，水洗和防水互斥。
> 选材不是选"最好的"，是选"最适合这个零件的"。
>
> 先问三个问题：
> 1. 这个零件最怕什么？（摔？弯？热？水？）
> 2. 哪个性能是绝对不能妥协的？
> 3. 其他性能可以接受到什么程度？
>
> 答完这三个问题，树脂就选出来了。
