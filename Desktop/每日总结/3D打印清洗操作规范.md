# 3D 打印清洗操作规范

> 清洗是最细节、最重要的问题。一步失误，轻则污染树脂，重则烧屏幕。

---

## 零、铁律

> ⚠️ **使用新设备/新机型时，必须先问实验室负责人或熟悉该设备的人。不要私自盲目操作。**
>
> 每台打印机脾气不一样，参数、操作细节、隐患点都不同。问了再动，不丢人；不问就干，坏了赔钱。

---

## 一、手套管理

| 规则 | 说明 |
|------|------|
| **两套丁腈橡胶手套** | 一套专用清洗（可沾树脂/酒精），一套保持干净用于其他操作 |
| **不要交叉** | 清洗手套碰过的东西，干净手套不再碰 |
| **拿平台只拿最上端** | 平台其他地方有残留树脂，碰到就污染手套 |

---

## 二、打印前检查

1. **拆开检查各零部件** — 确保没有残留树脂、没有松动
2. **重新扭紧** — 防止运行中松动移位
3. **平台下压确认** — 平台触碰屏幕时看是否平整贴合，确认无误再走
4. **确保所有部件清洁** — 一个脏点都可能导致失败

---

## 三、倒树脂规范

| 步骤 | 操作 |
|------|------|
| 1 | 倒树脂入一次性杯 |
| 2 | **擦干净瓶口（潦草口）** — 残留树脂干了会封死 |
| 3 | 擦干净后再盖回瓶盖 |

---

## 四、气泡处理

| 步骤 | 操作 |
|------|------|
| 1 | 树脂倒入料槽后，静置片刻让气泡自然上浮 |
| 2 | **用塑料铲沿边缘轻轻刮**，把气泡推到边边 |
| 3 | 确认料槽内无可见气泡后再开始打印 |

> 气泡不弄干净 = 打印件表面有坑洞/缺陷。塑料铲刮边边，简单有效。

---

## 五、清洗顺序（关键！）

```
先洗平台 → 再洗料槽
```

**为什么：** 平台不干净有树脂滴到屏幕上 = 屏幕报废。所以先处理平台，清干净了再碰料槽。

---

## 六、平台清洗

1. **先刮脊柱部分（上端连接轴）** — 残留树脂最多的地方
2. **倾斜拿出来** — 避免树脂滴到其他地方
3. **用塑料铲刮干净平台下部分** — 铲干净之后才移走
4. **不要泡平台** — 酒精长时间浸泡会掉漆/损伤涂层
5. **拿平台只抓最上端** — 避开树脂残留区域

---

## 七、屏幕和离型膜清洗

| ❌ 禁止 | ✅ 正确做法 |
|------|------|
| 酒精直接倒/喷屏幕上 | 酒精先喷到**无纺布**上 |
| 用纸巾擦（掉屑） | 用无纺布擦拭 |
| 用力擦拭离型膜 | 轻柔擦拭，离型膜很脆弱 |

⚠️ **酒精是溶剂，有腐蚀性。** 直接接触屏幕 = 短路风险。

---

## 八、打印成品清洗标准

| 状态 | 判定 |
|------|------|
| **干爽、不粘手** | ✅ 洗干净了 |
| **黏糊糊的** | ❌ 树脂没清洗干净，继续洗 |

> 以干爽不粘为唯一标准。摸上去黏 = 还有残留。

---

## 九、核心原则

```
手套不交叉 → 平台不滴液 → 酒精不直喷 → 成品不粘手
```

| 环节 | 口诀 |
|------|------|
| 手套 | 一套脏一套净，各干各的 |
| 倒树脂 | 瓶口擦干净再盖 |
| 清洗顺序 | 上（平台）→ 下（料槽）|
| 屏幕 | 酒精只走无纺布 |
| 成品 | 干爽不粘才算完 |
| 平台 | 不泡、倾斜拿、只抓头 |
