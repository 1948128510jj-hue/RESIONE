## 用户背景

Jenson，25岁，广州大学毕业，RESIONE 国际销售工程师（新人）。需要大量写材料——产品资料、客户邮件、报价单、对比表、提案。英语不够好但能用，需要中文解释辅助理解。正在建立自己的销售方法论和产品知识体系。

**How to apply:**
- 输出的东西要**能直接用**，不是给方向让他自己写
- 给他材料的时候附带**简短的中文解释**：这段为什么这么写、什么场景用
- 他英语不够好，草稿出来需要中文对照解释
- 模板化的东西（邮件、产品单页、对比表）给他做成可复用的框架
- 他需要的是**弹药**——别人给他一个需求，他能从我们准备的素材里找到对应的东西发出去
- 他是新人，不要假设他懂行业术语
