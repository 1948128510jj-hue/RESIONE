## 王雩赢销截拳道 核心技法

### 一、魂·道·术·器 四层框架

| 层 | 核心问题 | 要义 |
|------|------|------|
| **魂** | 你为什么做 | 感动客户——不是奉承，是亲人般真诚。不是来收钱的，是来帮他的。 |
| **道** | 方向对吗 | 先做人后做事。无信任不谈业务，无价值不谈价格。聚焦决策人。 |
| **术** | 具体怎么做 | 3P法则、五类客户应对、一分钟自我介绍、非工作时间试探。 |
| **器** | 能复制吗 | 标准化话术模板、每日动作清单、CRM追踪、团队训练体系。 |

### 二、销售三不急（铁律）

1. **不急着说话** — 先听客户说
2. **不急着推销** — 先弄懂需求
3. **不急着报价** — 先传递价值

> "急着说话、急着推销、急着报价——是销售三大死穴。"

### 三、3P面谈法则

- **P1 - Problem（问题）**：问清事实真相。专业选手问问题，普通选手急于推销。
- **P2 - Possible（可能性）**：分析可能方案，帮客户梳理解决路径。
- **P3 - Proposal（方案）**：以客户痛点为中心，提供全流程解决方案。

### 四、五类客户应对话术

| 客户类型 | 核心策略 | 话术方向 |
|----------|----------|----------|
| **爱讲价型** | 把焦点从"比价"转移到"体验价值" | "价格包您满意，市场透明不敢报高，您先体验感受" |
| **百般挑剔型** | 不反驳不争执，点透价值与价格关系 | "贵的东西除了贵没毛病，便宜的哪哪有问题" |
| **犹豫不下单型** | 用标杆案例打消顾虑 | "奔驰贵但买的人多，您要的是价值，我给同等品质最低价" |
| **懂行专业型** | 先认同再共情 | "您内行明白一分钱一分货，低价必质量缩水" |
| **贪便宜型** | 满足优惠心理→转化为转介绍 | "优惠可以给，但请介绍两位朋友" |

### 五、关键战术

**从上往下打**：直接找决策人（老板/董事长）。两点之间直线最短。拿下老板=拿到圣旨。

**3.125%概率法则**：100个好友 → 层层筛选 → 成交约3.125%。核心：正确面对失败，最大程度拜访，把时间花在最该花的人身上。

**非工作时间联系**：下班前半小时、晚饭后、周末联系——试探信任度和忠诚度。"把客户变成朋友，把朋友变成亲人。"

**一分钟自我介绍**：不是介绍产品，是告诉别人你有多优秀。让客户记住你、喜欢你、信任你、尊重你、愿意追随你。

**500万打印机案例**：局长拒绝推销→王雩提出借旧报纸研究→从"推销者"变"研究者"→最终成交并成为长期合作伙伴。面对拒绝，跳出产品，深度理解客户业务。

### 六、硬标准

- 每月100个客户是成功保障
- 每日5个以上拜访是行为基准
- 年收入÷平均客单价÷成交率÷可信度÷250天 = 每日必拜访数
