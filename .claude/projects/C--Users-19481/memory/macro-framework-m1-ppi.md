## M1-PPI 剩余流动性框架

### 核心公式

```
M1 同比增速 - PPI 同比增速 = 剩余流动性（"闲活钱"）

M1 = 市场上的活钱（现金 + 活期存款）
PPI = 工业品出厂价格，反映实体经济赚钱效应

M1-PPI > 0 → 闲活钱在积累 → 利好股市（尤其是科创/创业板）
M1-PPI < 0 → 闲活钱减少 → 利空股市
M1-PPI 扩大 → 加速上涨
M1-PPI 收窄 → 警惕风格切换或调整
```

### 内在逻辑

- **PPI 为负** → 实体企业靠降价卖货，老板不投资不扩张 → 钱不进实体 → 钱进股市
- **PPI 转正** → 实体开始赚钱 → 钱分流到实体 → 股市"闲活钱"减少
- **M1 扩大** → 央行放水，活钱变多 → 利好所有资产
- **M1-PPI 剪刀差** → 衡量"资产荒"的程度：差越大，越多人被迫进股市

### 四象限操作框架

| 情景 | M1-PPI 状态 | 策略 |
|------|-----------|------|
| **情景 1** | 维持当前点位 | 指数稳住，板块轮动，选赛道 |
| **情景 2** | 拐头向上 | 中小创全面牛，仓位转向科技/创业板 |
| **情景 3** | 拐头向下但不转负 | 存量博弈，指数震荡，风格切换 → 转高股息/顺周期 |
| **情景 4** | 拐头向下转负 | 中度调整，减仓 |

### 赛道选择逻辑

- **PPI 为负时**：逃离与 PPI 高度绑定的传统大盘蓝筹（制造业/重工业/中游加工），拥抱"与国内 PPI 绝缘"的资产——科创/创业板、高成长赛道（讲星辰大海/技术壁垒/全球产业趋势）
- **PPI 转正时**：钱回流实体，顺周期受益（有色/化工/资源品），科技"与 PPI 绝缘"的逻辑弱化
- **M1-PPI 收窄但 PPI 为正**：最利好的是"PPI 受益 + 独立逻辑"双线

### 选股启示

- 当前的"PPI 转正"是过去 12 个月最大的宏观变化
- 解释了为什么有色/化工/资源品涨、AI 高位震荡
- 应用：AI 赛道 PE 普遍 >50x 不值得追，PPI 受益的顺周期 + 独立逻辑是最优解
