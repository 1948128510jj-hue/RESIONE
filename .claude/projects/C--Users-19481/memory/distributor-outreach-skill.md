## distributor-outreach Skill V3

路径：`.claude/skills/distributor-outreach/SKILL.md`

### V3 vs V2：从"写邮件工具"升级为"外贸实战搭档"

| 维度 | V2 | V3 |
|------|----|----|
| 角色 | 销冠写邮件 | 外贸实战大师·全能搭档 |
| 用户定位 | 未定义 | 外贸新人（小白），3D打印树脂 |
| 业务政策 | 无 | 内嵌：不独家默认/MOQ 1kg/样品运费到付 |
| 输出范围 | 邮件 | 邮件+WhatsApp+领英+报价应答+样品沟通+价格谈判 |
| 额外能力 | 无 | 客户心理分析+回复构思+避坑提醒+实操思路 |
| 客户认知 | 一般 | 双轨：终端客户vs经销商关心点完全分开 |

### 固定业务政策（系统内嵌）
1. 经销权限：常规不独家，大客户可洽谈
2. 样品运费：默认到付
3. MOQ：1公斤

### 十大写作铁律
1. 经销商视角 2. 杜绝通用模板 3. 不贬竞品 4. 段落呼吸 5. 句式通俗
6. 结尾必有提问 7. 标题避开spam词 8. 区域风格区分 9. 预判融入 10. 场景区分

### 场景话术库
样品运费应答 / MOQ应答 / 不独家应答 / 价格异议应答

### 搭档能力清单（6项）
分析客户意图 / 构思回复话术 / 避坑提醒 / 开发跟进思路 / 专业应答 / 判断客户真实动机

**Why:** 用户是同一个人但身份变成了"被辅导的外贸小白"，Skill需要从"工具"升级为"搭档"——不只是输出文案，还要解释、判断、提醒、带路。

**How to apply:** 用户说任何外贸相关关键词 → 调用后判断是"出文案"还是"答疑/分析" → 出文案严格十铁律+三格式 → 答疑结合固定政策+客户认知给专业建议。
