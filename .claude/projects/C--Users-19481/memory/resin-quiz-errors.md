## 错题记录（2026-06-17）

### 错题 1：临时牙材料选择

- **题目**：牙科临时冠/桥，患者口内试戴
- **错误答案**：D01S
- **正确答案**：**RESIONE 产品线目前没有口腔内用材料**。D01S 是技工端高精度模型树脂（92D/5%伸长），不能放嘴里。全牙科线（D01/D01S/C01/GM01）都不具备 ISO 10993 生物相容性认证。
- **力学最接近的替代**：TH-MINI（但缺合规准入）
- **错误根源**：混淆了"牙科模型树脂"和"口内终用材料"——性能到了不等于法律能卖
- **关键词**：临时牙 ≠ 牙科模型，口内需要生物相容性认证

### 错题 2：无人机外壳材料选择

- **题目**：无人机外壳原型，能飞、轻、摔不碎
- **错误答案**：TH-MINI
- **正确答案**：**Anti-Impact**（46-98 J/m 冲击 + 26-27 MPa 拉伸 + 27-34 MPa 弯曲，产品册原文标注"无人机/机器人/RC"）
- **错误根源**：只看冲击强度（MINI 60-66 > Anti 下限 46），忽略了拉伸/弯曲强度（Anti 26-27/27-34 是 MINI 15/18.3 的近两倍）。无人机外壳需要**同时**满足抗摔（冲击）和结构刚性（拉伸/弯曲），缺一不可
- **关键教训**：选材必须同时看冲击 + 拉伸/弯曲两个维度，不能只看一个
- **防反**：伸长率 ≠ 刚性，拉伸强度和弯曲强度才是结构件硬指标
- **关键词**：无人机 → Anti-Impact（产品册写了），桌游战棋 → TH-MINI（产品册写了）

## 薄弱点总结

1. **牙科线定位不清**：不熟悉四款牙科产品的具体用途分工（模型 vs 铸造 vs 牙龈 vs 口内），需要重新梳理
2. **单指标决策倾向**：容易抓住一个突出参数（如冲击强度 60-66）就下结论，忽略其他维度的指标（拉伸/弯曲强度）可能已经低于使用门槛
3. **产品册原文记忆不够**：Anti-Impact 和 TH-MINI 的定位在产品册里写得很明确，但考场时没想起来
