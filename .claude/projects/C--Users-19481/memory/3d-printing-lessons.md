## 3D打印实操教训

### 屏幕损坏事故（2026-06-11）
- **原因**：树脂混入酒精，渗漏到屏幕，导致短路
- **教训**：
  1. 一定要清理料槽口
  2. 酒精是溶剂，不能直接接触电子产品
  3. 实在要用酒精，先喷到无尘布/无菌布上再擦拭

### Z轴高度
- 一般调到 5mm
- 贴地打（0mm）：一般用于实验，难拆卸，影响美观
- 贴地打通常不拆底

### 操作习惯
- 打印前看公司抖音官方号的讲解
- 细节决定成败，每一步都要检查

### 知识积累
- 主流软件的英文名称
- 各打印机机型的中英文名称
- 打印机各个部件的准确名称
- 不能模棱两可，必须准确记住

### 首层不粘底排查（2026-06-16）
- CL-TH 手机壳第一次没贴底，同参数第二次成功
- **Z轴零点不准是首因** — 刚性树脂间隙稍大就压不实
- **平台清洁** — 手指油印/上次残留膜必须酒精擦掉
- **树脂温度** — 刚倒出来冷的粘度高，料槽里温热后才正常
- 参数没变但结果不同 → 一定是物理问题，不是曝光问题

### 手机壳打印要点（2026-06-16）
- 摆法：背面朝上、内侧朝下、斜放 35-40°
- 支撑：中支撑杆 + 细接触点（0.45mm/0.2mm），疤痕藏内侧
- 大件平放 → 截面太大 → 离型力拉脱
- 选材：Anti-Impact（抗摔）> CL-TH（好看但脆）

### 支撑核心原则（2026-06-16）
- 刚细韧粗：刚性树脂接触点细好拆，韧性树脂接触点粗扛拉力
- 杆粗头细：支撑杆要够粗扛离型力，接触点要细好拆不留疤
- 固化前拆支撑 > 固化后拆
