## 印度牙科3D打印树脂市场调研

### 一、市场总览

- 市场规模：$112M-225M (2024-2025)，CAGR 16-37%
- 3D打印设备进口 CAGR 27.69% (2020-2024)，中国是第一大出口国
- 牙科lab数字化渗透率 FY25约55-63%，FY22仅28% — 三年翻倍
- 透明牙套病例 FY24 17,978 → FY25 22,678 (+26%)

### 二、竞品格局

**印度本土制造商：**
- Prevest DenPro (BSE上市)：FY25营收₹63Cr (~$7.5M)，净利₹18.2Cr（28.8%净利），127人。产品：Model / Model Pro / C&B Interim & Permanent / C&B Ceramic。有FDA/CE/USFDA/ISO13485认证。Model ~₹3,500/kg (~$42)
- WOL3D (NSE上市)：FY25营收₹48-49Cr (~$5.8M)，240人。自研Itech品牌树脂（标准/水洗/ABS/柔性），非牙科专用。标准树脂~₹2,000/kg。Shark Tank品牌效应。IPO募₹21.78Cr
- Protomont (私营)：₹2-5Cr，~10人。FiLAMONT品牌，非牙科专用。代理Elegoo/Anycubic/Creality。增长中

**国际品牌（进口至印度）：**
- eSUN DM100 (中国)：牙科模型树脂，印度零售₹3,999-4,999/kg (~$48-60)，线上分销(Robu.in, 3DBazaar)
- Jamg He Standard Plus (中国)：低价模型树脂，₹1,999/kg (~$24)
- ELEGOO Castable (中国)：铸造树脂，₹7,999/kg，Protomont经销
- MOLEGRID/Kexcelled (中国)：铸造树脂，₹8,499/kg
- NextDent/Formlabs (欧盟/美国)：全系列牙科树脂，高端价位

### 四、D01S定位

- 目标价位：中端偏上，FOB $27-35/kg (100kg+)
- 印度建议零售价：₹2,500-3,500/kg
- 优势：收缩率0.3% (vs eSUN 0.5%+)，硬度92D (vs Anycubic 80D)，三品牌全兼容
- 劣势：无CE/FDA认证（对中高端客户是减分项）
- 差异化：参数调好+打印测试文件+免费样品+技术支持

### 五、进入策略

1. 找经销商，不走直销（印度lab分散、决策链长、需要本地仓储）
2. 先搞定Harsh Juthani (Protomont) — 现成分销网络+缺牙科专用树脂
3. 同时联系Robu.in和3DBazaar (线上平台，已在售eSUN)
4. 免费样品策略：5份500g D01S + 打印测试文件 + 推荐参数表
5. 网易外贸通搜"dental resin"进口印度、量>100kg、反复进口的公司
