## PDF排版标准

以 `RESIONE_W1.pdf`（外贸第一周行动计划）为模板。

### 尺寸
- **A5**（148×210mm）— 手机直接看，不用缩放

### 配色（暖棕系，零蓝色）

| 元素 | 色值 | 用途 |
|------|------|------|
| 表头背景 | `#8b7355` 驼色 | 表格标题行 |
| 章节标题 | `#8b7355` 驼色 | H1标题 |
| 小标题 | `#6b5a3e` 深棕 | H2标题 |
| 封面标题 | `#4a3728` 深棕 | 封面大字 |
| 表格框线 | `#e8d5c4` 暖米色 | 表格网格线 |
| 斑马行 | `#faf7f2` 暖白 | 表格交替行 |
| 正文 | `#333333` 深灰 | 正文文字 |

### 字体
- 中文：微软雅黑（`C:\Windows\Fonts\msyh.ttc`）
- 字号：正文 10.5pt / 表格 9.5pt / 章节标题 16pt

### 表格规范
- 表头：暖棕背景 + 白色文字 + 居中
- 斑马条纹交替行
- 暖米色细网格线（0.3pt）
- 单元格上下留白 4pt

### 页面规范
- 左右边距 12-14mm
- 章节之间用暖棕分隔线
- 封面独立，章节之间不拆页过碎

### 生成工具
- Python reportlab
- `gen_pdf.py`（行动计划）/ `gen_d01_pdf.py`（产品对比）/ `gen_dental_pdf.py`（牙科手册）
- 所有 PDF 生成脚本统一用此配色

### 禁止
- 禁止使用任何蓝色系（包括深蓝、浅蓝、灰蓝）
- 禁止使用冷色调（绿、紫、青）
- 禁止纯黑文字（用 #333333 代替 #000000）
