When writing LinkedIn posts, messages, or captions for the user, output the text as clean, directly copyable blocks. Do NOT wrap them in markdown code fences (```) or add extra formatting. Just give the raw text with a separator line (---) above it. The user copies this directly into LinkedIn so it needs to be LinkedIn-ready: appropriate line spacing, no markdown, no emoji rendering issues.

**Why:** User copies these texts directly into LinkedIn and needs them to work without cleanup.
**How to apply:** Write LinkedIn text as plain blocks separated by ---. No ``` fences. Use LinkedIn-compatible formatting. Keep line breaks consistent with LinkedIn's editor.
