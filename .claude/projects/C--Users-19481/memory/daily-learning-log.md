用户每天下午 17:30-17:45 写当天复盘，内容包括：
- 今天打印了什么产品，效果怎么样
- 学了什么产品知识（当天那款）
- 发了多少封开发信，有没有回复
- 拍了什么视频素材

总结文件放在桌面，格式为每日 Markdown，方便积累。这个习惯已经写入第一周行动计划第十四节。

**Why:** 每天积累一点，一个月后回头看就知道自己走了多远。

**How to apply:** 每次新对话开始时提醒用户写昨天的复盘（如果还没写的话）。每周五汇总当周复盘。
