## 常见问题速查



### 颜色问题

- **TH-WW**: 灰色和火焰橙，没有透明色

- **WW123**: 6种颜色（海水蓝/水晶透/水晶紫/湖水绿/火焰橙/冷灰）

- **透明树脂**: G217(透明微蓝), F39T(透明柔性), C01(透明绿铸造), WW123(水晶透)

- **纯黑**: K+(纯黑注塑感), K(黑色), F69(黑色柔性), Anti-Impact(黑色)
