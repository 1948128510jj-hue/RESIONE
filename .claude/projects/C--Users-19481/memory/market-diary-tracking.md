## 复盘跟踪框架

**文件位置：** `C:\Users\Administrator\Desktop\A股美股复盘日记.html`（源文件 + PDF导出）

### 每日观察维度（12项）

1. **大盘指数** — A股五大指数(上证/深证/创业板/科创50/北证50) + 美股四大指数(SPX/NDX/DJIA/RUT)
2. **涨跌家数** — 涨/跌比、涨跌中位数、涨停/跌停数
3. **板块涨跌** — 领涨Top5 + 领跌Top5 + 各自驱动逻辑
4. **连板天梯** — 高度板 + 断板跟踪 + 晋级率 + 炸板率
5. **市场情绪** — 阶段定性(高潮/退潮/修复) + 核心锚定标的
6. **量能资金** — 成交额、主力净流向、北向资金、两融余额
7. **关键节点** — 连涨/连跌记录、压力/支撑位、宏观数据节点
8. **核心个股&龙头** — A股四大主线龙头 + 美股核心驱动个股 + 跨市场联动链
9. **赚钱效应/亏钱效应** — 按方向分列 + 杀伤力评级(🔴1-5) + 覆盖面%
10. **风险偏好** — 量化评分(0-100) + 表面vs实际偏好(美股)
11. **推荐策略** — 仓位建议 + 主线方向 + 回避清单 + 风控线
12. **确定性个股推荐** — ★★★★★/★★★★标的 + 逻辑+催化+展望+风险

### 更新时间

- **A股**：每个交易日 15:30 CST 后（收盘后30分钟）
- **美股**：每个交易日次日 08:30 CST 后（对应美东16:00收盘）

### HTML更新方式

每个交易日新增一个 `date-divider` + 对应的市场卡片/板块/情绪/策略区块，追加在上一日之后、下一日预览之前。

### 数据来源

公开市场信息：东方财富、同花顺、新浪财经、Yahoo Finance、Nasdaq.com、FXStreet、eOption
