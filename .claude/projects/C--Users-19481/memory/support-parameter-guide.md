## 支撑参数速查

### 按树脂类型

| 树脂类型 | 接触点直径 | 接触深度 | 密度 | 说明 |
|----------|:--------:|:--------:|:---:|------|
| **高韧性**（Anti-Impact, TH72, TH-MINI）| 0.7-0.8mm | 0.35-0.4mm | 60-65% | 韧性强，支撑粗了拆不动 |
| **刚韧均衡**（CL-TH, M58, K+, TH-BJD, Tough74 V2, G217）| 0.45-0.6mm | 0.2-0.3mm | 65-75% | 标准设置，脆断干脆 |
| **高刚性**（HT-Enduse, SP64, D01S）| 0.4-0.6mm | 0.15-0.25mm | 70-80% | 脆！太粗一拆就崩 |
| **柔性**（F69, F39, F80）| 0.8-1.0mm | 0.4-0.5mm | 55-65% | 太软，支撑要够粗才拉住 |
| **水洗**（WW123, WW-ABS, TH-WW）| 0.5-0.7mm | 0.25-0.3mm | 65-75% | 水洗树脂略软，取中间值 |

### 按模型类型

| 模型类型 | 支撑策略 | 注意 |
|----------|----------|------|
| 手办/雕像 | 中支撑为主，接触点细，可见面不放支撑 | 外观优先 |
| 功能件 | 中+重支撑，允许内侧有疤痕 | 尺寸准确优先 |
| 微缩模型 | 轻支撑，0.3mm 接触点 | 拆支撑不能崩细节 |
| 大面积平板 | 斜放 30-45°，底面均匀铺支撑 | 平放 = 必翻车 |
| 空心件 | 必须加排气孔！| 离平台最近最高点打孔 |

### 核心原则

- **刚细韧粗**：刚性树脂接触点细（好拆不崩），韧性树脂接触点粗（扛得住扯）
- **杆粗头细**：支撑杆要够粗扛离型力，接触点头要细好拆
- **固化前拆支撑**：所有树脂通用
- **可见面不放支撑**：把疤痕藏内侧/底部

### CL-TH 手机壳参考值

| 参数 | 值 |
|------|:---|
| 接触点直径 | 0.45mm |
| 接触深度 | 0.2mm |
| 支撑类型 | 中支撑 |
| 密度 | 70% |
| 摆放 | 背面朝上，内侧朝下，斜 35-40° |
