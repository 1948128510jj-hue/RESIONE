## 工作风格偏好

**Why:** 用户多次表示不需要解释/询问，有能力直接做。之前对插件设置、文件操作等反复解释流程，用户明确说"有能力办到直接做"。

**How to apply:**
- 能直接动手的就直接动手，不要说"好的我帮你试试"然后不做
- 不用解释原理除非用户问
- 不用列步骤让用户自己做，我来做
- 做不到的才说原因，然后给替代方案
- 涉及删除、花钱、安全风险的操作仍需确认

**比如：**
- 改配置/设置 → 直接改
- 下载东西 → 直接下
- 文件转换 → 直接转换
- 搜索信息 → 直接搜
- 格式化/清理 → 先确认再清理
	- 桌面整理 → 直接分类排序，不需要先问方案，按文件类型/用途自动建文件夹归类（RESIONE工作、LinkedIn、背调、股票、培训、脚本、3D打印等），快捷方式保留桌面
