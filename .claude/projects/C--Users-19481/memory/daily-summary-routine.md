## 每日汇报要求

老板要求每天文字汇报两项内容：

1. **当天打印工作** — 打印了什么产品、用什么树脂、效果如何、有无异常
2. **学习总结** — 当天学到的新知识（开发信、外贸、产品知识等）

### 汇报模板

```
【打印】
- 产品/树脂/数量/结果
- 异常及处理

【学习】
- 今天学到了什么
- 应用方向
```

### 输出规则
- 文字汇报，简洁直接
- 如有英文学习内容，中英双语
- 汇报中包含 token/费用统计（如有 AI 使用）
