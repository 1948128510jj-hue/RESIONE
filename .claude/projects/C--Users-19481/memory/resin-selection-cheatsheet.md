## 树脂选型速查表

### 按应用场景

| 场景 | 首选 | 备选 | 原因 |
|------|------|------|------|
| **手机壳** | Anti-Impact | — | 93%伸长率+98J/m冲击，唯一扛得住摔的 |
| **潮玩/机甲/积木** | CL-TH | M58 | 9色可选，刚韧均衡 |
| **BJD/可动手办** | TH-BJD | Tough74 V2 | 不黄变，17-34%伸长|
| **透明特效件** | G217 | WW123(水晶透) | G217高透可打磨，钻孔攻丝 |
| **微缩棋子(Warhammer)** | TH-MINI | TH72 | 45.6%伸长，60-66J/m冲击，耐把玩 |
| **功能零件(钻孔攻丝)** | M58 | K+, G217 | 22-42J/m冲击，可机加工 |
| **纯黑注塑感** | K+ | K | 0.30%低吸水，户外可用 |
| **柔性/橡胶手感** | F69(黑) | F39(白), F39T(透) | 71A, 222%伸长 |
| **超软牙龈** | GM01 | F80 | 50A级柔软 |
| **高温模具** | HT-Enduse | — | 140°C耐温，97D硬度 |
| **日常/教学/新手** | WW123 | WW-ABS | 水洗方便，12mPas近水流 |
| **牙科模型** | D01 | D01S | 高精度，耐磨耐刮 |

### 关键指标速查

| 指标 | 看什么 | 代表型号数值 |
|------|--------|-------------|
| **抗冲击(摔不烂)** | 缺口冲击强度 J/m | Anti-Impact: 46-98 |
| **韧性(掰不坏)** | 断裂伸长率 % | F69: 222%, Anti-Impact: 93% |
| **刚性(硬不硬)** | 邵氏硬度 D | HT-Enduse: 97D, SP64: 95D |
| **抗弯(折不断)** | 弯曲模量 MPa | CL-TH: 1940-2060 |
| **耐磨** | 弯曲强度 MPa | HT-Enduse: 97.88 |
