## 整体结构（A4，1页）

### 头部（无边框表格）
- **名字**：左上，26pt Bold #4a2c17（深暖棕）
- **英文名**：名字下方，10pt #c4956a（浅暖棕）
- **基本信息**：9pt #999999（灰）和 #333333（正文），左对齐
- **求职意向**：11pt Bold #8b5e3c（中暖棕），浅米色背景条 #FAF3E8
- **照片**：右上，标准竖版证件照（蓝色底正装），约2.5-2.8cm宽×3.5-3.8cm高

### 段标题（每个模块标题）
- 12pt Bold #4a2c17
- **左边深棕竖条装饰**（6pt粗，颜色#4A2C17，距文字4pt）
- 前后间距约6pt/2pt

### 教育经历
- 学校+学位+日期：**同一行**
- 学校：10pt Bold #4a2c17
- 学位：9pt #8b5e3c，分隔符|用8pt #999999
- 日期：7.5pt #999999
- 课程/荣誉：8pt #333333，缩进0.5cm

### 个人优势
- 用●深棕圆点（#4a2c17），不加粗，缩进0.4cm
- 正文9pt #333333
- 4条

### 工作经历
- 公司名：10pt Bold #4a2c17，独立一行
- 职位：9pt #8b5e3c，独立一行
- 日期：7.5pt #999999，**右对齐**（使用右对齐tab stop）
- 内容：●深棕圆点 + 9pt #333333正文，缩进0.5cm

### 专业技能
- 标签：8.5pt Bold #8b5e3c，格式【标签】
- 内容：8.5pt #333333

## 配色系统（商务深蓝系 — 海外销售专用）
- 深藏蓝 #1a2a3a（标题、公司名、圆点、竖条）
- 中蓝 #2c5f8a（职位、学位、技能标签、求职意向）
- 浅蓝灰 #6b8fad（英文名）
- 正文 #333333
- 灰色 #888888（日期、分隔符、辅助信息）
- 浅蓝底 #eef3f7（求职意向背景条）
- **暖棕系仅V5使用，V7+统一深蓝商务风**

## 字体
- 全局微软雅黑（Microsoft YaHei）
- 英文自动匹配系统字体

## 排版规则
- 页边距：上0.9cm 下0.6cm 左1.5cm 右1.5cm
- 行距：1.0倍
- 无表格线（仅头部用无边框表格实现照片右对齐）
- 所有正文9pt，课程/荣誉8pt，技能8.5pt
- 简历必须控制在一页

- 日期右对齐：在职位行使用右对齐tab stop（位置9072 twips）
