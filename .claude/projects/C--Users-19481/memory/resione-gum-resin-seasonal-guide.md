## F80 vs GM01 季节性选型

用户 Jenson 的口述经验：

- **夏季（热）**: 优先用 **F80**（牙龈粉色）。天气热，树脂流动性本身就高，F80 粘度 2180 mPa·s 在高温下表现更稳定。
- **冬季（冷）**: 仅用 **GM01**（牙龈粉）。冷天树脂流动性下降变稠，GM01 粘度 3200 mPa·s 在低温下反而可控。

**Why:** 温度影响光敏树脂粘度。F80 中粘度(2180)在夏天不会过稀，GM01 高粘度(3200)在冬天不会过稠。选反了会出打印问题——夏天用 GM01 太稀挂不住，冬天用 F80 太稠流不动。

**How to apply:**
- 向牙科客户推荐牙龈树脂时，根据客户所在地气候和车间温度给出建议
- 温控打印室的客户可忽略此建议，按柔软度需求直接选（GM01=50A 超软, F80=64A 中软）
- 可在商务沟通中展示 RESIONE 提供不同粘度选择以覆盖不同生产环境
