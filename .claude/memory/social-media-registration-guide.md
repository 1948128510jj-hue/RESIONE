## 注册所需

1. **苹果手机** — 需绑定WhatsApp；需美国App Store ID下载WhatsApp/翻墙App/Facebook/Ins
2. **新电话号码** — 用于账号验证
3. **两个邮箱** — Gmail/QQ/163各备（Ins随机适配，一个被封换另一个）
4. **DUO Mobile** — 双重验证码

## 养号节奏

| 周 | 操作 |
|:--:|------|
| 第1周 | 只浏览，不关注不私聊不发帖 |
| 第2周 | 适量关注账号，发帖 ≤3篇 |
| 第3周+ | 逐步叠加 |

## 企业运营建议

- 用公司号而非个人号注册App Store ID，方便后续更新
- CL-TH产品宣传重点：全球唯一多色坚韧桌面树脂
- 发布内容以"彩色免涂装"为卖点切入
- 优先在Instagram/YouTube/TikTok发布产品视频
- 可联系美国3D打印博主做产品测评引流
