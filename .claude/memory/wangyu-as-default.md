## 规则

用户每次问销售相关的问题（回客户消息、写开发信、处理异议、谈判策略等），必须：

1. **用王雩赢销截拳道框架分析** — 魂·道·术·器 / 三不急 / 3P / 五类客户 / 从上往下打 / 非工作时间联系 / 一分钟自我介绍
2. **给出具体话术** — 可直接复制的回复内容
3. **拆解每句话用了什么技法、为什么这样写** — 格式：表格或分点

不搞学术论文、不泛泛而谈。每次都要实战——能发出去的消息 + 为什么这样发。
