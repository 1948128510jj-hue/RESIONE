# 3D打印行业生态图谱

## 一、设备厂商
造机器的。消费级 Elegoo/Anycubic/Creality 年出货百万台级；专业级 Heygears/Uniformation/联泰；国际 Formlabs/Prusa。
- 合作价值：机器买家名单 = 树脂潜在客户
- 风险：部分厂商推官方树脂捆绑，是竞合关系

## 二、材料经销商
中间商赚差价。一级代理签协议拿最低价有区域保护；二级批发加价卖零售；淘宝/京东/亚马逊是终端流量入口。
- 一个经销商手里几十个代打和设计工作室客户，搞定一个=覆盖一片市场

## 三、打印服务商（代打）
客户发模型→服务商打印→收加工费。
- 手办代打：动漫模型+上色
- 牙科代打：牙模/导板
- 工业手板：产品验证
- 珠宝代打：蜡模铸造
- 批量生产：潮玩文创
- 树脂耗材大户，月用几十公斤

## 四、终端用户
金字塔底，量最大最分散。
- 个人爱好者：1-2瓶/月，B站/淘宝评价驱动
- 设计师/工作室：5-10瓶/月，要特定颜色/手感/不黄变
- 手办/潮玩：几十公斤/月，要便宜稳定
- 牙科：NMPA合规，要认证/生物相容
- 制造业：要交期/批次一致性

## 获客路径
- 个人/设计师 → 内容种草（B站测评、群聊推荐）
- 手办/牙科 → 关系+样品试打
- 制造业/批量 → 经销商覆盖
- 所有 → 一个行业群就是以上四类人的缩影
