## 核心观点

看新闻20年才发现的真相：赚钱的生意都写在新闻联播里。新闻联播不会告诉你买什么股票，但它告诉你钱在往哪里流。

## 政策 → 投资的三层映射

```
政策文件/新闻联播 → 资金流向（钱往哪去）
                 → 产业方向（什么被扶持）
                 → 瓶颈环节（扶持链条中哪段最窄）
```

## 当前政策框架（2026年6月）

### "六张网" 新型基础设施
- 水网 + 新型电网 + 算力网 + 新一代通信网 + 城市地下管网 + 物流网
- 2026年投资 > 7万亿
- 新型电网"十五五"投资 > 5万亿
- 算力网"十五五"投资 > 万亿级
- **关键交叉点：算电协同（算力网 × 新型电网）**

### "两新"政策
- 设备更新：2026年2000亿（已下达1851亿）
- 消费品以旧换新：2026年1875亿
- 6月底前全部下达

### AI + 能源双向赋能
- 四部门联合行动方案（2026年4月）
- 51个高价值场景清单发布
- 算电协同24次写入文件
- 绿电直供13次写入文件

## 使用方法

每次分析股票前，先扫近期新闻联播/政策 → 锁定资金流向 → 再沿产业链找瓶颈。
和 M1-PPI 框架、Serenity 瓶颈理论构成三层决策体系。
